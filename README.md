Sim Questões é uma plataforma web de banco de questões voltada para vestibulares (com foco em ENEM e UEMA), criada para organizar, padronizar e escalar a construção de uma base robusta de exercícios. O projeto foi pensado para transformar um acervo de questões em um sistema estruturado, com taxonomia clara (matéria → conteúdo → tópico) e um fluxo consistente de importação, facilitando tanto a manutenção do banco quanto a evolução do produto ao longo do tempo.
O repositório concentra o frontend construído com Vite + React + TypeScript, utilizando shadcn/ui e Tailwind para uma interface minimalista, rápida e fácil de evoluir, além da integração com Supabase para autenticação, persistência de dados e rotinas relacionadas ao banco de questões. Também inclui documentação essencial para o pipeline de cadastro, como o padrão de importação via JSON, além de arquivos de referência de IDs e instruções que garantem consistência na inserção de questões.
Objetivos principais do projeto:
Centralizar e organizar um banco de questões de forma escalável
Garantir padronização de estrutura (IDs, taxonomia e formato de importação)
Facilitar automações de ingestão de questões (pipeline de cadastro)
Manter o produto extensível para novos exames, matérias e recursos
Stack / Tecnologias:
Vite + React + TypeScript
shadcn/ui + Tailwind CSS
Supabase (auth + banco + integrações)
Documentação de apoio para importação e referência de IDs
Este repositório serve como base para evolução contínua do produto, com foco em organização, consistência de dados e uma experiência rápida e objetiva para estudo e gestão de questões.
